:checked是什么？

:checked CSS 伪类选择器表示任何处于选中状态的radio(), checkbox () 或(“select”) 元素中的option HTML元素(“option”)。

用户通过勾选/选中元素或取消勾选/取消选中，来改变该元素的 :checked 状态。
